Este proyecto consiste en el desarrollo de una API REST utilizando **Spring Boot** para la gestión de alumnos, profesores, materias e inscripciones.

## Funcionalidades
- Gestión de Alumnos, Profesores y Materias (CRUD completo)
- Inscripción de alumnos en materias con borrado en cascada
- Documentación automática con Swagger

##  Tecnologías utilizadas
- Java / Spring Boot
- MySQL (Base de datos)
- Maven (Gestor de dependencias)
- Git (Control de versiones).
